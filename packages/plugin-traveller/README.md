# @telemetry/plugin-traveller

Fair Use Policy layer for the Traveller setting (Spec §1). This package implements
`packages/engine`'s plugin API against Traveller-specific terms and content; the engine
itself contains zero setting content and zero Traveller terms (INV-1).

The Mongoose Fair Use disclaimer and `NOTICE.traveller` are an owner decision
(`docs/tasks/M0-00.md`) and have not landed yet. Do not distribute this package
until that task closes.
