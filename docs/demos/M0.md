# M0 demo — the spine

## What this proves

This local walkthrough proves the M0 vertical slice using shipped content and UI: a private hotseat handoff, a JSON-scripted four-beat turn, phase transitions committed through the single ledger writer, public facts appearing in the ship's log, and projections updating from those facts. It is a technical spine demo, not yet a playable trade game.

## Clean-checkout setup

Requirements: Node 20 or newer, pnpm 9, and a locally installed Chrome or Chromium browser.

```sh
pnpm install --frozen-lockfile
pnpm test
pnpm test:integration
pnpm lint
pnpm lint:content
pnpm build:stub
pnpm build:pages
pnpm test:e2e
```

All commands must pass. `pnpm test:e2e` boots the production bundle under the GitHub Pages base path and closes it automatically.

## Launch the local demo

```sh
pnpm build:pages
pnpm demo:m0
```

Open `http://127.0.0.1:4173/telemetry-engine/`. Keep the launch command running until the walkthrough is complete.

## Five-minute walkthrough

1. Confirm the shared-screen regions are visible behind a **Hand the device to Zhan** interstitial: status bar, four-beat phase track, main panel, and ship's log.
2. Select **I am Zhan**. Confirm Zhan's private view contains `agenda.actionTaken`; confirm that private fact does not appear in the public ship's log.
3. Confirm **DOCKSIDE** is active and the main panel begins, “Dockside systems are open.”
4. Select **Advance demo turn**. Confirm **COMMS** becomes active and `cargo.loaded` appears in the public ship's log.
5. Select **Advance demo turn**. Confirm **TRANSIT** becomes active and the main panel reports a jump plotted from Regina to Vantage.
6. Select **Advance demo turn**. Confirm **ARRIVAL** becomes active, the main panel reports eighteen crates paid against a manifest of twenty, and `jump.plotted` appears in the public ship's log.
7. Select **Advance demo turn**. Confirm the cycle returns to **DOCKSIDE**, `sale.settled` appears in the public ship's log, and funds read `Cr169,200`.

## Pass criteria

- Every clean-checkout command exits successfully.
- The demo completes all seven walkthrough checks without a blank screen, console-visible failure, or missing announcement.
- Private `agenda.actionTaken` remains absent from the public ship's log.
- The turn returns to DOCKSIDE with the sale fact and updated funds visible.
