# M2-06 — Social sim policies, lineups, and metrics

## Tests first
- property: comms, accusation, confrontation, and vote policies read only the actor's scoped view
- integration: L3 witch-hunt and L5 all-dirty lineups complete campaigns deterministically
- unit: misattribution, false-accusation, envelope-burn, detection, and ship-survival metrics use harness ground truth, never policy peeking

## Done when
- the existing table-driven policy module covers all social situations and the sim exports the M2 metrics
- threshold sweeps report distributions needed for the retro

## Do not
- no learning, lookahead, LLM calls, or hidden-ledger access inside a policy
- do not tune by weakening INV-5/10's zero-unique-attribution property
