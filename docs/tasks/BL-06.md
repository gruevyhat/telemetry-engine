# BL-06 — Send the Mongoose Fair Use Policy notification email

## Context
`NOTICE.traveller` (repo root) landed the required on-site disclaimer per Mongoose
Publishing's Traveller Fair Use Policy FAQ item 1(b). FAQ item 1(c) separately requires
direct notification by email to sales@mongoosepublishing.com. That email is drafted, not
sent — an implementer session should not dispatch real correspondence to a third-party
publisher on the owner's behalf. Spun out of `M0-00` so the license-decision task isn't
blocked on an owner-only send.

## Tests first
- n/a — owner correspondence, not a code task

## Ready-to-send text
**To:** sales@mongoosepublishing.com
**Subject:** Traveller Fair Use Policy — web site notification

> Hello,
>
> Per your Traveller Fair Use Policy FAQ item 1(c), I'm notifying you that I have a
> Traveller-adjacent web site at this URL:
>
> https://gruevyhat.github.io/telemetry-engine/
>
> This is Telemetry Engine, a non-commercial, open-source fan project: an event-sourced
> tabletop game referee tool. It includes an optional Fair Use Policy compatibility layer
> (packages/plugin-traveller) for use with the Traveller game, letting the tool consume
> Traveller sector data via a one-time, user-driven import from travellermap.com. The tool
> itself contains no Traveller rulebook text — all game content is original dice math and
> original entries, per your Fair Use Policy's guidance on derivative works.
>
> The required site-wide disclaimer is posted at:
> https://github.com/gruevyhat/telemetry-engine/blob/main/NOTICE.traveller
>
> Thank you,
> Graham Horwood
