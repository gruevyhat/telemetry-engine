# M2-03 — Agenda content format and independent setup deal

## Prerequisite
M2-00 must record owner approval for the `AgendaAction` schema before this schema-changing task begins.

## Tests first
- property: one independent Bernoulli draw per player; odds 0 deals none and odds 1 deals all
- property: positive and negative results both commit `agenda.dealt` plus public verifying commitments linked to the campaign's sole prior seed commitment
- content-lint: tiers, selectors, actions, access, claimant references, and weights are valid and referentially complete

## Done when
- agenda/deck schemas and loader implement Spec §10.2; setup emits proposals and the interpreter commits the whole deal
- each player receives a private objective presentation even when their agenda result is negative

## Do not
- no global traitor-count draw and no guarantee that anyone is disloyal
- do not expose result, tier, action menu, or packet contents on the shared screen
