# M2-15b — App assembly, sim wiring, and milestone demo

## Prerequisite
M2-15 (`docs/handoffs/M2-15-1.md`): the integration and e2e capstone tests already prove
`dealAgendas`/`queueCommsAction`/`resolveConfrontation`/`assembleBlackBoxArtifact` and the
pairing/comms/disconnect UI compose. This card is the remaining gap that session found and
flagged rather than built through: nothing has assembled M2-08..14's standalone, unit-tested
pieces into a runnable app, and `packages/sim` never wired M2 mechanics past bot decision-making.

## Scope
- `packages/sim`: wire `dealAgendas`/`queueCommsAction`/`advanceCommitted`/`resolveConfrontation`/
  `assembleBlackBoxArtifact`/`verifyBlackBoxArtifact` into a real M2-aware campaign runner so
  `runSocialCampaign` (or its successor) actually drives comms windows and confrontations for
  lineups L2/L3/L5, feeding `metrics.ts`'s already-waiting misattribution/detection machinery.
- `content/`: author a production agenda-deck JSON (real content, validated by
  `packages/content-lint`, not the inline test fixtures M2-11..15 used).
- `packages/ui-shared` / `packages/ui-phone`: enough session wiring (host build pairs phones via
  the real trystero adapter, phone build receives snapshots and renders `CommsScreen`, shared
  screen renders `PairingCard`/`DisconnectPanel`/`ConfrontationPanel`) to make a three-seat
  comms-to-accusation scene actually clickable from a clean checkout. This does not need to be
  the full M2 feature set live in the UI -- it needs to cover the walkthrough below.
- `docs/demos/M2.md`: clean-checkout commands plus a walkthrough matching Plan §5's M2 exit demo
  -- pair at least two phones, deal independent agendas at published odds, complete a timed comms
  window with two colliding actions, force a disconnect and resume or fall back to hotseat,
  surface an incident, carry or fail an accusation vote, and finish by verifying the black-box
  preimages. Follow `docs/demos/M1.md`'s shape (What this proves / Clean-checkout setup / Launch /
  Walkthrough / Pass criteria / Known gaps / Run record).

## Tests first
- property/sim: lineups L2, L3, and L5 complete a full agenda/comms/confrontation campaign with
  zero unique twin attribution (INV-5/10) and every secret draw verifying against the black box
- integration: the assembled session wiring drives one comms window and one confrontation through
  real transport-encoded messages (not direct interpreter calls), matching M2-11's protocol shapes
