# M0-05 — RNG named streams

## Tests first
- property: stream independence — an extra draw on stream A leaves stream B's sequence unchanged
- property: same seed + same draw sequence = same outputs cross-platform
- unit: stream derivation from campaign seed is stable across restarts

## Done when
- xoshiro128 core; derive(streamName) API; draw-count tracking per stream (commit-reveal will need indices at M2)

## Do not
- Do not expose a global RNG; every draw names its stream
